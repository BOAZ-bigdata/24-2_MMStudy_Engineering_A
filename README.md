# 2024 BOAZ 엔지 A조 멘멘 스터디

## 룰

- `기본적으로 자료를 정리해오기 필수`
  - 깃허브 리포에 자료 올려두기
  - 발표자를 미리 정하고 5~10분 내외로 발표하기
    - 기본적으로 멘토 1 멘티 1 조합
      - 멘티 2명도 가능
    - **하고 싶은 주제를 먼저 선정하는 방식**
    - 남는 자리는 랜덤 선정
- `질문 2개 필수`
  - 스터디 시간에 자유 토론
- `과제`
  - **멘토가 주는 미션 - 경훈아 부탁해**
- `스터디 시간`
  - 매주 수요일 오후 5시 ~ 오후 6시
